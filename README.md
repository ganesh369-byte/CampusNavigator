# Campus Navigator

A C++ application that finds the shortest path between campus locations.

## Features

- Add roads between buildings
- Find shortest path
- Calculate distance
- Uses Dijkstra Algorithm

## Technologies

- C++
- OOP
- Graphs
- STL
- Priority Queue

## Project Structure

```text
src/
data/
README.md
```

## How to Run

```bash
g++ src/main.cpp src/Graph.cpp -o navigator

./navigator
```
